# TCL Center 01 — Electro Template (Embedded Images Fix)

This build embeds all visible website images directly inside `index.html`.

## Upload to GitHub Pages
Replace the old files in the repository root with:

- `index.html`
- `favicon.svg`
- `robots.txt`
- `sitemap.xml`
- `.nojekyll`

No `assets` folder is required. After committing, wait 1–3 minutes and hard refresh with `Ctrl + F5`.
